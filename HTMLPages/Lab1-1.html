Web Page Elements

When you view a web page in your brower, you are seeing the interpreted code that you type into your text editor. Web pages are made up of text which is called "Marked Up", or formatted using Hyper Text Markup Language or HTML. The HTML codes, known as tags, are placed around the text that you want to format. In most cases, when formatting text, you will have both an opening and closing tag to indicate where the formatting should start and where it should end.

Content on a web page is contained inside elements. There are basically two tyoes of elements, block and inline. This content is being written inside a paragraph element, which is black element because there will be a space above and below the paragraph essentially blocking it offfrom other content. However this content is in a strong element which is inline. Inline elements do not separate their content. Block elements will effect a pages layout, where inline elements will not.

There are all different types of elements that go to make up a web page. The elements used on this page is the HTML, which is also called the root element, the head element, body element, heading element, paragraph element, and the strong in line element. The elements used on this page, other than the strong element, are the minimum that any web page should include.
